# 💻 Laptop Price Predictor

## 📌 Project Description

Laptop Price Predictor is a Machine Learning web application that predicts the estimated price of a laptop based on user specifications such as brand, RAM, processor, storage, operating system, GPU, screen resolution, and weight. The application uses a trained Random Forest Regression model to provide accurate price predictions through a simple Flask web interface.

---

## ✨ Features

- Predict laptop prices instantly
- User-friendly interface
- Supports multiple laptop brands
- Uses Machine Learning for prediction
- Built with Flask
- Responsive web design

## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/sampadawale217-crypto/Laptop-Price-Predictor.git
```

### 2. Go to Project Folder

```bash
cd Laptop-Price-Predictor
```

### 3. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open Browser

```
http://127.0.0.1:5000
```
